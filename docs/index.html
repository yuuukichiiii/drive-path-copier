Drive Path Copier

Google Drive で開いているフォルダのパスを、Windows のエクスプローラー形式でワンクリックコピー。

G:\共有ドライブ\プロジェクト\フォルダ名
GitHub リポジトリ プライバシーポリシー

概要

Drive Path Copier は、ブラウザの Google Drive で現在開いているフォルダの場所を、 Windows エクスプローラーで使えるパス文字列（例：G:\共有ドライブ\…）に変換し、 クリップボードにコピーする Chrome 拡張機能です。Drive for Desktop でマウントしたドライブの ファイルを、エクスプローラーや他アプリですばやく開きたいときに役立ちます。

主な機能

  • 共有ドライブ・マイドライブの両方に対応
  • Google アカウントごとにドライブ文字（G:、H:、I: など）を設定可能
  • 取得するのはフォルダ名と親フォルダ情報のみ（読み取り専用）
  • 設定はブラウザ内にのみ保存し、外部サーバーへは一切送信しない

使い方

  • Google Drive でフォルダを開く
  • ツールバーの拡張機能アイコンをクリック
  • 「Windows パスをコピー」を押す
  • 初回のみ Google の認証画面で許可
